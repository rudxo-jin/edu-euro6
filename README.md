# 정비사 교육용 웹앱 - 마인드맵 생성기

정비사 교육을 위한 **이미지 기반 마인드맵 자동 생성** 웹 애플리케이션입니다. 기존 마인드맵 이미지를 업로드하면 AI가 구조를 분석하여 인터랙티브한 웹 마인드맵으로 자동 변환해줍니다.

## 🎯 핵심 기능

### 📸 이미지 기반 마인드맵 생성
- **이미지 업로드**: JPG, PNG, GIF 등 다양한 이미지 형식 지원
- **자동 구조 분석**: AI를 통한 마인드맵 트리 구조 자동 인식
- **실시간 변환**: 분석된 구조를 바탕으로 인터랙티브 마인드맵 자동 생성
- **정확도 높은 텍스트 인식**: OCR 기술을 활용한 정확한 텍스트 추출

### 🗺️ 인터랙티브 마인드맵
- **드래그 앤 드롭**: 노드를 자유롭게 이동하여 레이아웃 조정
- **실시간 편집**: 노드 간 연결선 추가/제거
- **줌 및 네비게이션**: 확대/축소, 미니맵을 통한 전체 구조 파악
- **시각적 구분**: 계층별 색상 및 크기 자동 적용

### 📚 정비사 교육 특화
- **자동차 정비 도메인**: 엔진, 브레이크, 전기, 변속기 시스템 등
- **체계적 학습 구조**: 중심 주제에서 세부 항목으로 확장되는 구조
- **교육 효과 극대화**: 시각적 학습을 통한 이해도 향상

### 💾 데이터 관리
- **원본 이미지 보존**: 업로드한 원본 이미지와 생성된 마인드맵 함께 저장
- **JSON 형식 저장**: 표준 형식으로 데이터 저장 및 공유
- **프로젝트 복원**: 저장된 파일을 통한 완전한 작업 환경 복원

## 🛠️ 기술 스택

### Frontend
- **React 18** + **TypeScript**: 타입 안전성과 최신 React 기능 활용
- **React Flow**: 전문적인 마인드맵 인터페이스 구현
- **Tailwind CSS**: 모던하고 반응형 UI 디자인
- **Lucide React**: 일관성 있는 아이콘 시스템

### Build & Development
- **Vite**: 빠른 개발 서버와 최적화된 프로덕션 빌드
- **PostCSS**: CSS 전처리 및 최적화
- **ESLint**: 코드 품질 관리

### AI & Image Processing (향후 통합 예정)
- **Google Vision API**: 고정밀 OCR 및 텍스트 인식
- **AWS Textract**: 문서 구조 분석
- **Custom AI Model**: 마인드맵 특화 구조 인식

## 📖 사용법

### 기본 워크플로우
1. **이미지 준비**: 마인드맵이 그려진 이미지 파일 준비
2. **업로드**: "마인드맵 이미지 업로드" 버튼으로 이미지 선택
3. **분석**: "이미지 분석하기" 버튼 클릭하여 AI 분석 시작
4. **확인**: 왼쪽 패널에서 분석 결과 확인
5. **편집**: 오른쪽 패널에서 생성된 마인드맵 확인 및 조정
6. **저장**: 완성된 마인드맵을 JSON 파일로 저장

### 고급 기능
- **노드 위치 조정**: 생성된 노드를 드래그하여 최적 위치로 이동
- **연결선 편집**: 노드 간 새로운 연결 추가 또는 기존 연결 수정
- **프로젝트 관리**: 저장/불러오기를 통한 여러 마인드맵 관리
- **초기화**: 새로운 프로젝트 시작을 위한 전체 초기화
